Ejercicios Resueltos de Cálculo Integral

Ejercicio 1

Calcular la siguiente integral:

∫ (3x²) dx

Desarrollo:

Aplicamos la regla de la potencia:
∫ xⁿ dx = (xⁿ⁺¹)/(n+1) + C

∫ 3x² dx = 3 · (x³ / 3) = x³ + C

Resultado:

x³ + C

Ejercicio 2

Calcular la integral definida:

∫02 (2x) dx

Desarrollo:

∫ 2x dx = x²

Evaluamos en los límites:
x² |02 = 2² − 0² = 4

Resultado:

4

Ejercicio 3 (Aplicación – Área bajo la curva)

Función: f(x) = x²

Intervalo: [0, 2]

Integral:

∫02 x² dx

Desarrollo:

∫ x² dx = x³ / 3

(2³ / 3) − (0³ / 3) = 8 / 3

Área bajo la curva:

8/3 unidades cuadradas
