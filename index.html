Temario de Cálculo Integral

1. La derivada de una función

La derivada representa la razón de cambio instantánea de una función respecto a su variable independiente. Geométricamente, es la pendiente de la tangente. Permite analizar comportamiento local, máximos, mínimos y puntos de inflexión.

2. Modelos basados en la derivada y análisis de errores

Permite construir aproximaciones lineales de funciones y analizar el error entre valores reales y aproximaciones. Aplicable en ingeniería, física y economía.

3. Notación suma (Σ)

Representa sumas finitas de manera compacta. Es base para las sumas de Riemann y la definición de integrales.

4. Áreas bajo la curva

El área bajo una curva se calcula mediante integrales definidas y representa acumulación continua de cantidades.

5. Integral definida y sus propiedades

Calcula el área neta sobre un intervalo. Cumple propiedades como linealidad y aditividad.

6. Teorema del Valor Medio para integrales

Existe al menos un punto en el intervalo donde la función toma su valor promedio, útil para física y estadística.

7. Teorema Fundamental del Cálculo

Relaciona derivadas e integrales como procesos inversos. Facilita el cálculo de áreas mediante antiderivadas.

8. Método de sustitución

Permite simplificar integrales complejas mediante cambio de variable.

9. Integración por partes

Integra productos de funciones usando la regla del producto inversa.

10. Integrales con potencias de seno y coseno

Se aplican identidades trigonométricas para simplificar integrales con funciones trigonométricas elevadas a potencias.

11. Áreas entre curvas

Se obtiene integrando la diferencia entre dos funciones, útil para problemas de geometría y física.

12. Volúmenes de revolución

Calcula volúmenes al rotar una curva alrededor de un eje.

13. Longitud de arco

Permite medir la longitud de una curva usando integrales.

14. Trabajo y energía

El trabajo realizado por una fuerza variable se calcula mediante integrales.

15. Centro de masa

Determina el punto de equilibrio de un cuerpo o región mediante integrales ponderadas.

16. Integrales impropias

Se usan cuando los límites son infinitos o la función es discontinua, para calcular áreas y acumulaciones especiales.

17. Aplicaciones en economía

Se utilizan para calcular costos, ingresos y utilidades acumuladas mediante integrales.

18. Integrales en física

Se usan para calcular velocidad, aceleración, energía y trabajo de fuerzas variables.

19. Series y sumas infinitas

Relacionadas con integrales impropias y aproximaciones de funciones, base para Fourier y Taylor.

20. Métodos numéricos de integración

Métodos como trapecio o Simpson para aproximar integrales de funciones complejas.

21. Aplicaciones en ingeniería

Integrales para estructuras, hidráulica, circuitos y control de procesos.

22. Integrales múltiples

Integrales dobles y triples para calcular volúmenes, superficies y centros de masa 3D.

23. Cambio de variables en integrales múltiples

Usa coordenadas polares, cilíndricas o esféricas para simplificar integrales en 2D y 3D.

24. Aplicaciones avanzadas

Flujo de fluidos, probabilidad, ecuaciones diferenciales y modelado matemático usando integrales.
