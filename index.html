Ejercicios Resueltos de Cálculo Integral

Ejercicio 1

Calcular la siguiente integral:

∫ (3x²) dx

Desarrollo:

Aplicamos la regla de la potencia:
∫ xⁿ dx = (xⁿ⁺¹)/(n+1) + C

∫ 3x² dx = 3 · (x³ / 3) = x³ + C

Resultado:

x³ + C

Ejercicio 2

Calcular la integral definida:

∫02 (2x) dx

Desarrollo:

∫ 2x dx = x²

Evaluamos en los límites:
x² |02 = 2² − 0² = 4

Resultado:

4

Ejercicio 3 (Aplicación – Área bajo la curva)

Función: f(x) = x²

Intervalo: [0, 2]

Integral:

∫02 x² dx

Desarrollo:

Temario de Cálculo Integral

1. La derivada de una función

La derivada representa la razón de cambio instantánea de una función respecto a su variable independiente y se interpreta geométricamente como la pendiente de la recta tangente.

2. Modelos basados en la derivada y análisis de errores

Las derivadas permiten construir modelos matemáticos aproximados y analizar el error entre valores reales y aproximaciones lineales.

3. Notación suma

La notación sigma se utiliza para representar sumas finitas y es fundamental para la definición de las sumas de Riemann.

4. Áreas bajo la curva

El área bajo una curva se calcula mediante integrales definidas y representa una aplicación geométrica del cálculo integral.

5. Integral definida y sus propiedades

La integral definida calcula el área neta en un intervalo y cumple propiedades como linealidad y aditividad.

6. Teorema del Valor Medio para integrales

Este teorema indica que existe un punto donde la función toma su valor promedio en el intervalo.

7. Teorema Fundamental del Cálculo

Relaciona la derivada con la integral, mostrando que son procesos inversos.

8. Método de sustitución

Permite simplificar integrales mediante el cambio adecuado de variable.

9. Integración por partes

Se utiliza para integrar productos de funciones mediante una fórmula específica.

10. Integrales con potencias de seno y coseno

Se aplican identidades trigonométricas para facilitar la integración.

11. Áreas entre curvas

El área entre dos curvas se obtiene integrando la diferencia de funciones.

12. Volúmenes de revolución

Permite calcular volúmenes de sólidos generados por rotación.

13. Longitud de arco

La longitud de una curva se calcula mediante una integral específica.

14. Trabajo y energía

El trabajo realizado por una fuerza variable se obtiene mediante integrales definidas.

15. Centro de masa

Representa el punto de equilibrio de una región o cuerpo.

16. Integrales impropias

Se utilizan cuando los límites son infinitos o la función es discontinua.
